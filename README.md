# MIUI IME Unlock

解锁 MIUI 全面屏优化限制

## 测试环境

LSPosed v1.2.2(API 93.0)

MUI 12.5

Android 11(R,API 30)

## 使用方法

Xposed API Version >= 93

在作用域勾选需要解除限制的输入法即可，小米定制版也要勾上
